Knowledgebase

adi-knowledgebase-cli, rust, cli, knowledge-management

Overview

  • CLI interface for ADI Knowledgebase
  • Commands: add, query, approve, clarify, conflicts, ask, show, delete, link, orphans, status

Commands

  • kb add --user-said "..." - Add knowledge with user statement
  • kb query - Search the knowledgebase
  • kb approve - Approve a node (confidence = 1.0)
  • kb clarify - Request clarification
  • kb conflicts - Show conflicts
  • kb ask - System asks user a question
  • kb show - Show node details
  • kb delete - Delete a node
  • kb link - Link two nodes
  • kb orphans - Show orphan nodes
  • kb status - Show status

Dependencies

  • adi-knowledgebase-core - Core library
  • lib-cli-common - Common CLI utilities
  • clap - CLI parsing
