Overview

  • Debug micro app for Metal shader development
  • Focus: GPU text rendering, shader compilation, Metal API exploration
  • macOS only (Apple Silicon + Intel)
  • License: BSL-1.0

Purpose

  • Isolated environment for Metal shader experimentation
  • GPU text rendering research (fontdue, rustybuzz)
  • Not part of main release cycle - development tool only

Dependencies

  • metal: Apple Metal API bindings
  • cocoa/objc: macOS framework bindings
  • winit: Window management
  • fontdue/rustybuzz: Font rasterization and shaping
